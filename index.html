Third Year Project

A group project composed of three people. The objective for this project was to find a dataset and to analise the affect the data from the dataset have on each other. Juniper Notebook was used as the IDE and Python was the language we used to discover data points. The dataset we chose was Internet connections in households from the years 2006 and 2011. We used Python to filter and create graphs from the data. The graphs were used to identify trends or changes in the data when comapring against each other. Calculations for correlation coefficient, central tendancy and variabillity were also used for this project. In the end we found out that there was a steady rise of internet connections in homes from 2006 to 2011, as expected.

Second Year Project

This project was a group project composed of three people. The objective for this project was to create a website using Java, Scala and HTML. My team agreed to create a movie type website which has access to all the latest movies. There were three iteration for this project each with its own major objectives.

Iteration one was to figure out the structure of the website and starting the process of coming up with a template. Iteration two was to build the website with the ideas from iteration one and adding feature like a basket or login system. Iteration three was to polish the website and make it look somewhat proffessional, any final features like password encryption, security for logins, user accounts, registrations and contacts were also configured from iteration two.

Finally to explain how the website functions. New users can register onto the website from the registration page. They can then login and select any movie they wuld like to purchase. A purchase requires the user to enter their banking information which will then be saftley stored onto the database and movie delivered to the user digitally. The admin account can edit, add and delete movies. The admin also has a control over the prices of movies.
